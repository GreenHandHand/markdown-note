# 编程相关概念

常见语言分类：

- 静态类型语言(Dynamic)：编译时检查类型
    - 错误检测：早(在执行前)
    - 代码灵活性：低
    - 性能：通常更快
    - 例如：Rust, Java, C#
- 动态类型语言(Static)：运行时检查类型
    - 错误检测：晚(在执行时)
    - 代码灵活性：高
    - 性能：可能会更慢
    - 例如：Python, JavaScript, Ruby
- 弱类型语言(Weak)：对数据类型的赋值约束少/无约束
- 强类型语言(Strong)：对数据类型的赋值进行规则约束

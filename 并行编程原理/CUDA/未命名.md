# CUDA C++ 基础

CUDA 设计是为了：
1. 充分利用 GPU 的并行性，将其用作通用计算
2. 提高性能

CUDA C++ 是指：
1. 基于工业基础的 C++
2. 是 C++ 的允许异构编程的拓展
3. 一种直观地、用于管理设备(也就是 NVIDIA GPU)、内存之类的 API

> [!info] 在本节中，主要介绍的是 CUDA 的 C++ 接口，这是目前运用最广泛的接口。其他的一些语言也支持 CUDA，例如 Fortran, Python, Matlab 等。

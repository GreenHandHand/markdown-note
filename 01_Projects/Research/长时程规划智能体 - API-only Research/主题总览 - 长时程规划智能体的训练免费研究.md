# 长时程规划智能体的训练免费研究

## 主题概述
这个主题聚焦于 **Long-horizon planning agents**，并且限定在 **training-free、API-only** 的研究设定下。

核心问题是：
- 长链路任务中如何保持 plan 的稳定性
- 出错后如何进行 recovery / replanning
- 如何减轻 semantic drift
- 如何在不做 fine-tuning、RL training、world model learning 的前提下提升 agent 的长期规划能力

## 我对主题的最新总结
目前这组材料最合适的统一主题是：

**长时程规划智能体（Long-horizon Planning Agents）的训练免费控制与恢复机制**

更具体地说，当前主线已经从宽泛的 agent planning 问题，收敛到：

**失败后的局部恢复（local repair）是否优于 full replanning**

这条线很清晰，因为它串联了：
- semantic drift
- recovery cost
- artifact reuse
- dependency-aware replanning

## 当前最核心的方法主线
当前最明确、最成型的方向是：

### Plan Splicing / Dependency-Scoped Local Repair
核心思想是：
当某一步失败后，不直接 full replan，而是先定位最小的失效子图（invalid subgraph），然后只修补这部分 plan，并尽量保留：
- 已完成步骤
- 未受影响的 plan structure
- 可复用的 artifacts

这是这个主题当前最值得优先推进的方案。

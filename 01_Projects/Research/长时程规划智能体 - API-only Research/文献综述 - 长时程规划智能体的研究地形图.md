## 中文总结
这份文献综述主要梳理了 **Long-horizon planning agents** 领域的研究路线，重点包括：

### 1. Hierarchical Planning + Task Decomposition
主要思路是让 high-level planner 负责子目标拆分，让 low-level executor 负责具体动作执行。
代表方向包括：HiPlan、L2M2、GLIDER、ReAcTree。

### 2. Process Supervision + Credit Assignment
研究重点是如何把最终任务成败分摊到中间步骤，解决 sparse reward 和 delayed credit assignment 问题。
代表方向包括：HICRA、Hierarchical Preference Learning、LOOP。

### 3. Error Recovery + Self-Correction
关注 agent 在执行过程中如何检测错误、诊断原因、恢复执行，而不是失败后直接重来。
代表方向包括：ReSeek、PALADIN。

### 4. World Models + Planning
通过 learned world model 在 latent space 中进行规划，但通常依赖训练。
这与当前 training-free、API-only 的限制形成对比。

## 关键研究空白
当前比较值得关注的 gap 有：
- credit assignment
- recovery / replanning
- semantic drift
- evaluation beyond success rate
- partial observability under uncertainty

## 我的提炼
如果限制在 **training-free, API-only**，最值得优先研究的不是“更强的训练框架”，而是：
- 更聪明的 replanning 触发机制
- 更局部、更低成本的恢复策略
- 更可靠的 failure attribution

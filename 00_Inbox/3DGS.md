# 3DGS

任务：基于图像的 3D 场景重建。

> [!note] Image based 3D scene reconstruction

## 传统三维重建技术

传统方法基于多视图几何理论，具有直观、可视化、清晰的结构表达和强兼容性的优点。

- sfm 方法：生成稀疏点云，并通过匹配图像间的特征点准确估计相机位姿。

传统方法的缺陷：

1. 效率低：依赖大量高重叠图像和复杂的空间加密过程，难以高效建模大规模场景。
2. 编辑和维护成本高。模型的修改需要手动操作定点或者重新映射。
3. 表达复杂结果和弱纹理区域的能力弱。
4. 对光照的适应性差。
5. 动态变化不足，生成的模型是静态的。

## 基于 NeRF 的三维重建

2020 年提出 NeRF (神经辐射场)，任务是做新视角的合成，并实现了摄像机基本的逼真的新视图合成。NeRF 可以简要概括为**用一个 MLP 去隐式学习一个静态 3D 场景**，实现复杂场景的任意心视角合成。

以任意的相机位置 + 朝向作为输入，经过训练好的神经网络进行体渲染，就可以得到任意视角的图片。

## 3DGS

3DGS 论文主要提出了三个内容：

1. 使用 3D Gaussians 作为灵活的、富有表达能力的场景表示方式。
2. 提出一个优化方法来优化 3D Gaussians 的参数——坐标、透明度、各向异性协方差矩阵和球谐函数 (SH)。
3. 一种实时渲染方法，利用 GPU 实现的快速排序算法。

### 可微 3DGS

3DGS 算法从 SFM 生成的点云开始。

使用 3D Gaussians，这是一种可微、可以通过 $\alpha$ 混合的方式投影到 2D 的表示方法。

区别于传统方法，传统方法假设每一个点都是一个具有法线向量的小圆。但是使用 SFM 生成的点云很难测量法线，而且 SFM 生成的点云是一种充满噪声的信息，从中估计法线是非常有挑战性的。

3D Gaussian 不需要估计法线。一个 3D Gaussian 使用一个 3D 协方差矩阵 $\Sigma$ 和一个由世界坐标定义的中心点 $\mu$ 进行定义：

$$
G(x) = e^{-\frac{1}{2}x^{T}\Sigma^{-1}x}
$$

之后这个 $G(x)$ 会使用 $\alpha$ 进行混合用于渲染。

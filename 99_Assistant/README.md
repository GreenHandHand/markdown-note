# 99_Assistant - 助手任务笔记区

这个目录专门存放助手的任务笔记。

整理规则：
- 按任务记录，而不是只按日期写流水账
- 每个任务单独一篇笔记
- 文件名必须能直接看出主题
- 优先使用格式：`YYYY-MM-DD - 任务主题.md`

适合记录的内容：
- 你交代给助手的任务
- 助手做过的调查、整理、修改、总结
- 关键结论、结果与后续建议

示例：
- `2026-03-24 - 创建助手笔记区与固定记录规则.md`
- `2026-03-24 - 整理 n8n 学习案例.md`
- `2026-03-24 - 检查 Obsidian 仓库结构与笔记分布.md`

约定：
- 不把图片/素材放在这里
- 需要时可以持续更新同一任务笔记
- 如果旧笔记标题含糊，应改名或拆分
